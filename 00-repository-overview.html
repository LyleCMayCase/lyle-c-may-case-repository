Repository Organization

The Lyle C. May Case Repository has been organized into collections, each serving a specific purpose. Together, these collections guide readers from the Supplemental Motion to the publicly available materials supporting its legal claims.

Each collection has been designed to help readers locate specific categories of information, including court filings, source materials, media coverage, repository downloads, and supporting documentation. Together, these collections provide a structured pathway through the repository and its contents.

The Collections

  • Start Here introduces the repository and explains how it is organized.
  • Supplemental Motion contains the complete motion and a section-by-section guide to its legal claims.
  • Repository Downloads provides downloadable copies of major repository documents.
  • Source Verification Archive contains the publicly available authorities cited throughout the Supplemental Motion.
  • Media highlights publicly available news coverage, interviews, and related reporting.
  • Updates documents significant additions and developments as the repository continues to grow.

Designed for Easy Navigation

Each collection has been organized to function independently while remaining connected to the larger repository. Readers may explore the collections in any order and return to previous sections at any time.
